>[!note] Basic RISC-V Relational Operations

|     Operation     | High-Level (C) |                      Description                       |
| :---------------: | :------------: | :----------------------------------------------------: |
| slt(i/u) x1 x2 x3 |  x1 = x2 < x3  | Sets `x1` to 1 if `x2` is less than `x3`, 0 otherwise. |
|                   |                |                                                        |
